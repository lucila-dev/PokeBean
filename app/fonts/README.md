# Pokemon font for the title

**PokemonSolid.ttf** is the **Pokemon Solid** logo-style font (the one that looks like the official Pokémon logo lettering). It is used for the "PokeBean" brand in the nav and on the home page. Source: [GuangchuangYu/meme](https://github.com/GuangchuangYu/meme) (inst/fonts/pokemon).

To use a different font, replace this file and the copy in `public/fonts/PokemonSolid.ttf`, or update the paths in `app/layout.tsx` and `app/globals.css`.
